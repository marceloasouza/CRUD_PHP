Instruções

===========================

O sistema contem as pastas views, model, controllers e banco.

Separado baseando na arquitetura MVC. Na pasta Views contem toda a parte visual do sistema, juntamente com a index.
Pasta Model contem o arquivo de criação de conexão com o banco e todas as chamadas para realização de insert, edit e delete.
Já na pasta controllers deixei todas as funções de CRUD e validação.

Na pasta banco, estão os arquivos referente ao banco de dados utilizado no sistema.

O sistema contem uma área restrita, para que o usuário com privilégios, já cadastrado no sistema, entre com o seu email e senha e tenha acesso a lista de usuários cadastrados, podendo alterar informações ou até mesmo deletar o usuário.
